# Token Utility

## Overview

BlockAI is designed with a future utility token model that powers the platform's credit system, governance, and incentive mechanisms.

> ⚠️ **Note:** The BlockAI token has not been launched. This section outlines the planned token utility model.

## Planned Utility

### 1. Credit Purchases

The primary utility of the token is purchasing **AI credits** within the platform:

- AI Chat messages
- Smart contract generation and auditing
- NFT generation
- Wallet analysis and transaction decoding

### 2. Premium Access

Token holders gain access to premium features:

- Unlimited AI queries
- Priority response times
- Advanced market signals
- Extended chat history persistence

### 3. Staking Rewards

Token staking provides:

- Discounted credit rates (up to 50% off)
- Governance voting power
- Revenue sharing from platform fees

### 4. Governance

Token holders can vote on:

- Feature prioritization
- New AI model integrations
- Fee structure changes
- Partnership decisions

## Token Distribution (Planned)

| Allocation               | Percentage | Purpose                                    |
| ------------------------ | ---------- | ------------------------------------------ |
| Community & Rewards      | 30%        | Daily rewards, referrals, airdrops         |
| Team & Advisors          | 15%        | 2-year vesting with 6-month cliff          |
| Development Fund         | 20%        | Platform development and AI infrastructure |
| Ecosystem & Partnerships | 15%        | ChainGPT, exchange partnerships            |
| Liquidity                | 10%        | DEX and CEX market making                  |
