# Incentive Model

## Current Incentive System

BlockAI uses a **points + credits** dual system to reward engagement and fund AI usage.

### Credits (AI Currency)

- **Sign-up bonus:** 20 free credits on account creation
- **Usage cost:** 1–3 credits per AI action
- **Replenishment:** Purchase via payment or earn through engagement

### Points (Loyalty Rewards)

Points are earned through platform engagement:

| Action                     | Points Earned |
| -------------------------- | ------------- |
| Daily Reward Claim (Day 1) | 5 PTS         |
| Daily Reward Claim (Day 2) | 10 PTS        |
| Daily Reward Claim (Day 3) | 15 PTS        |
| Daily Reward Claim (Day 4) | 20 PTS        |
| Daily Reward Claim (Day 5) | 30 PTS        |
| Daily Reward Claim (Day 6) | 40 PTS        |
| Daily Reward Claim (Day 7) | 50 PTS        |
| AI Chat Message            | 1 PTS         |
| Referral Sign-up           | 10 PTS        |

**Full 7-day streak = 170 PTS per cycle**

### Daily Rewards System

- Users can claim escalating point rewards every 24 hours
- Consecutive days build a streak (up to 7 days)
- Missing a day resets the streak to Day 1
- After Day 7, the cycle restarts

### Referral Program

- Each user gets a unique referral link
- When a referred user signs up, both parties earn points
- Referral tracking is persistent via the backend database

## Future Incentive Plans

### Token Conversion

When the BlockAI token launches, accumulated points will convert to tokens based on a defined conversion ratio. This rewards early adopters who engaged with the platform during the pre-token phase.

### Creator Rewards

Users who generate popular smart contracts, NFT templates, or AI workflow templates may earn additional rewards through a creator incentive pool.

### Staking Multipliers

Token stakers will receive multiplied point earnings:

- 1,000 tokens staked → 1.5x points
- 5,000 tokens staked → 2x points
- 25,000 tokens staked → 3x points
