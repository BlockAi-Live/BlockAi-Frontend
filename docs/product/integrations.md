# Integrations

BlockAI connects with best-in-class Web3 infrastructure providers.

## AI Provider

### ChainGPT

**5 API products integrated:**

| Product                  | Integration | Purpose                     |
| ------------------------ | ----------- | --------------------------- |
| General Chat AI          | REST API    | Crypto-native AI assistant  |
| Smart Contract Generator | REST API    | Natural language → Solidity |
| Smart Contract Auditor   | REST API    | Automated security analysis |
| NFT Generator            | REST API    | Text-to-image generation    |
| Crypto News              | REST API    | AI-curated news feed        |

[ChainGPT Documentation →](https://docs.chaingpt.org)

## On-Chain Data

### Etherscan V2 API

Provides raw blockchain data for wallet analysis and transaction decoding:

- Account transaction lists
- Token transfer history
- Account balances
- Contract source code and ABI

[Etherscan API Docs →](https://docs.etherscan.io)

## Market Data

### CoinGecko API

Powers all market-related features:

- Real-time prices for 10,000+ coins
- Historical chart data (1D–Max)
- Market cap, volume, and rankings
- Price change percentages

[CoinGecko API Docs →](https://www.coingecko.com/en/api)

## Wallet Infrastructure

### ThirdWeb

Handles all wallet connections and on-chain reads:

- **Supported wallets:** MetaMask, Coinbase Wallet, Rainbow, Rabby
- **Chains:** Ethereum Mainnet, Base
- **Features:** Balance queries, contract deployment (coming)

[ThirdWeb Documentation →](https://portal.thirdweb.com)
