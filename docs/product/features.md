# Features

## AI Chat

- **Dual-model support** — ChainGPT (crypto-native) and BlockAI 3.0 (Gemini-powered)
- **Streaming responses** — Word-by-word typewriter effect for natural conversation
- **Markdown rendering** — Code blocks, tables, lists formatted natively
- **Credit tracking** — Live credit counter with cost-per-message indicator
- **Points system** — Earn points for each interaction

## Market Analysis

- **Real-time data** — Powered by CoinGecko API (price, volume, market cap, rank)
- **Interactive charts** — 24H, 7D, 30D, and Max timeframes with smooth area charts
- **Scrolling ticker** — Marquee-style coin cards with live selection
- **AI reports** — One-click generation of market reports, alpha finder, and technical analysis
- **Search** — Instant search across all listed coins

## Chain Scanner (Wallet Intelligence)

- **Wallet analysis** — Paste any Ethereum address for a full profile
- **Transaction history** — Complete tx list with AI-powered explanations
- **Transaction decoder** — Paste a tx hash to see what happened in plain English
- **Risk assessment** — AI evaluates wallet behavior patterns
- **On-chain data** — Powered by Etherscan V2 API

## Smart Contracts

- **AI Generation** — Describe any contract in natural language, get production-ready Solidity
- **AI Auditing** — Paste contract code for automated security analysis
- **Syntax highlighting** — Full Prism-powered code display
- **Copy & export** — One-click copy for use in Remix or Hardhat

## NFT Generator

- **Text-to-image** — Describe your NFT vision, ChainGPT generates the artwork
- **Instant preview** — See generated art immediately
- **Download-ready** — Save artwork for minting on any platform

## Dashboard Features

- **Wallet tracking** — Connect MetaMask, Coinbase, Rainbow, or Rabby via ThirdWeb
- **AI News Feed** — Curated crypto news from ChainGPT
- **Daily Rewards** — 7-day streak system with escalating point rewards (5→50 PTS)
- **Demo Mode** — Toggle to preview the dashboard with realistic mock data
- **Watchlist** — Top coins with live sparkline charts and 24h change
- **AI Signals** — On-demand market intelligence (click to reveal)
- **Activity Feed** — Collapsible log of all AI interactions

## Authentication

- **Three methods** — Email/password, OAuth (Google, GitHub, Twitter), wallet connect
- **JWT-based** — Secure 24-hour tokens with background validation
- **Auto-logout** — Graceful session handling on token expiry
- **Access gateway** — Beta features gated behind access codes

## Referral System

- **Invite codes** — Generate shareable referral links
- **Point rewards** — Earn points when referred users sign up
