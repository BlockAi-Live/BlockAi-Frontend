# Product Overview

BlockAI is a unified Web3 intelligence platform with six core modules, each designed to solve a specific workflow within the crypto ecosystem.

## Module Map

| Module              | Purpose                                                  | Status     |
| ------------------- | -------------------------------------------------------- | ---------- |
| **Dashboard**       | Centralized command center with portfolio, news, rewards | ✅ Live    |
| **AI Chat**         | Crypto-native AI assistant (ChainGPT + Gemini)           | ✅ Live    |
| **Market Analysis** | Real-time charts, prices, AI reports                     | ✅ Live    |
| **Chain Scanner**   | Wallet analysis + transaction decoding                   | ✅ Live    |
| **Smart Contracts** | AI contract generation + auditing                        | ✅ Live    |
| **NFT Generator**   | Text-to-image NFT creation                               | ✅ Live    |
| **Smart Alerts**    | On-chain notifications (price, wallet, gas)              | 🔜 Q2 2026 |

## Dashboard

Your command center. At a glance you see:

- Wallet balance with USD conversion
- AI credit balance
- Live crypto news powered by ChainGPT
- 7-day daily rewards streak
- Market watchlist with sparkline charts
- AI market signals (on-demand)
- Recent activity log
- Notification feed

## Pricing

BlockAI uses a **freemium credit system**:

| Tier              | Credits                     | Price       |
| ----------------- | --------------------------- | ----------- |
| **Free**          | 20 credits on signup        | $0          |
| **Daily Rewards** | 5–50 points/day via streaks | Free        |
| **Premium**       | Unlimited                   | Coming soon |

Each AI action costs 1–3 credits depending on complexity.
